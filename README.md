## Короткое описание проекта

### Русский
**PDF/JPG Converter** — утилита для взаимной конвертации файлов PDF и JPG. Сканирует папку `files`, преобразует `.pdf` → `.jpg` и `.jpg` → `.pdf`, сохраняя исходные файлы. Если в папке уже есть пара файлов с одинаковым именем и разными расширениями, они пропускаются.
